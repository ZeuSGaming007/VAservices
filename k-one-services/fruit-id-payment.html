About Fruit ID Payment Services

We assist with the payment processes associated with Fruit ID registrations. Below are the required documents for these services.

Required Documents for Fruit ID Payment Services

Service Type Required Documents
New Fruit ID Registration
  • Aadhaar Card
  • Proof of Identity (Voter ID, Passport)
  • Proof of Address (Utility Bill, Ration Card)
  • Bank Account Details
Fruit ID Payment Processing
  • Existing Fruit ID Number
  • Proof of Payment (Receipt or Bank Statement)
  • Contact Information
Fruit ID Renewal Payment
  • Existing Fruit ID Number
  • Aadhaar Card
  • Proof of Payment (Receipt or Bank Statement)
